# VieNeu-TTS Landing Page Specification

This document outlines the complete requirements for building a modern, visually striking landing page for **VieNeu-TTS**, a Vietnamese Text-to-Speech model with instant voice cloning, 24 kHz speech generation, offline capability, and 1000 hours of curated training data.

---

## 1. Overview
The landing page introduces **VieNeu-TTS** as a next-generation Vietnamese TTS system designed for creators, developers, and businesses. The page must convey technological sophistication and the model's readiness for real-world applications.

---

## 2. Design Style
- Dark or gradient background (purple/blue neon tone)
- Subtle animations and motion effects
- Waveform visuals, glowing orbs, sound-based elements
- Clean, modern typography
- Strong visual hierarchy and spacious layout

---

## 3. Page Structure

### A. Hero Section
- Headline: **"VieNeu-TTS — Giọng nói Việt, chân thật đến từng hạt âm."**
- Subtitle: **"Instant voice cloning. 24 kHz. Hoàn toàn offline."**
- Action Buttons:
  - Try Demo
  - GitHub
  - Facebook (Author)
- Background animation: interactive waveform or glowing audio orb

### B. Key Highlights
Three feature blocks:
1. **1000 giờ dữ liệu tuyển chọn** – phát âm tự nhiên, ổn định
2. **Voice cloning tức thì** – chỉ cần 3–5 giây audio
3. **Song ngữ Việt–Anh** – code-switching mượt

### C. Audio Demo
- 10 sample voices (Male/Female, North/South accent)
- Each voice as a card with waveform animation
- Play button and optional voice upload area

### D. Core Features
- Natural Vietnamese pronunciation
- Fully offline inference
- CPU/GPU support, GGUF soon
- High-fidelity voice cloning
- Real-time synthesis
- Production-ready API/UI

### E. Model Comparison
Comparison between VieNeu-TTS-140h and VieNeu-TTS-1000h:
- Dataset size
- Pronunciation quality
- Cloning accuracy
- Code-switching
- Speed

### F. Video Showcase
- Embedded Facebook Reel installation guide

### G. Custom Voice Training Pricing

#### A. Cleaned + Transcribed Data
- Under 10h: **300 USD**
- 10–30h: **700 USD**

- Over 50h: Custom Quote

#### B. Raw Audio Data (includes cleaning + transcription)
- Under 10h: **500 USD**

#### Includes
- Commercial license

> Tip: Preparing data according to the official guidelines can reduce costs by **40–50%**.

### H. Technical Specification
- Backbone: Qwen 0.5B
- Codec: NeuCodec (ONNX + quantized)
- Context: 2048 tokens
- Watermark enabled
- Dataset: VieNeu-TTS-1000h
- Python API + CLI + Gradio UI
- Fully offline capability
